Local AI video captioning for macOS. Drop a video, get a captioned MP4 — no cloud, no API keys, no usage fees.

**Features**
- Per-word highlighting with champagne gold pill effect
- Powered by OpenAI Whisper (large-v3) running entirely on-device
- Roboto font, white text, black outline, drop shadow
- Supports MP4, MOV, MKV, AVI, WEBM

## Requirements

- macOS (Apple Silicon recommended)
- [Homebrew](https://brew.sh)

Everything else (whisper-cpp, ffmpeg-full, the Whisper model) is downloaded automatically on first launch.

## Development

```bash
git clone <repo>
cd subburnin
npm install
npm start
```

## Build DMG

First add your icon:
```bash
# Place a 1024×1024 PNG at assets/icon.png, then:
npm run make-icon   # generates assets/icon.icns
```

Build (requires Apple Developer ID for signing):
```bash
npm run build
```

Build without signing (for local testing):
```bash
npm run build:unsigned
```

Output is in `dist/`.

## Project Structure

```
src/
  main.js              Electron main process
  preload.js           contextBridge for main window
  setup-preload.js     contextBridge for setup window
  services/
    config.js          Read/write ~/.subburnin/config.json
    setup.js           First-run dependency installer
    whisper.js         whisper-cli transcription → JSON segments
    ffmpeg.js          Audio extraction + caption burning
    converter.js       Whisper segments → styled ASS subtitles
  renderer/
    index.html         Main drop-zone UI
    app.js             Main window logic
    setup.html         First-run setup UI
    setup-app.js       Setup window logic
    styles.css         Dark theme
assets/
  icon.png             Your 1024×1024 app icon (not committed)
  icon.icns            Generated by npm run make-icon (not committed)
scripts/
  build-icon.sh        PNG → icns converter
```

## Config

Settings are stored in `~/.subburnin/config.json`:

```json
{
  "whisper_model_path": "~/.subburnin/ggml-large-v3.bin",
  "whisper_threads": 4,
  "output_dir": ""
}
```

`output_dir` defaults to the same folder as the input video if blank.
